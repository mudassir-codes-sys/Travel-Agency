# Travel Agency Landing Page (React)

This project is a **responsive landing page** for a travel agency, built with **React**. It includes **signup and login pages** with backend authentication.  

## Features

- Fully **responsive** landing page (desktop, tablet, mobile)
- **Signup/Login pages** with backend authentication
- Modern **UI/UX design** 
- Built with **React** and **Tailwind CSS**
- Easy to **customize** colors, text, and layout
